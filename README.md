# 🐾 宠物外观版本测试跟踪器

一个多人协同的宠物外观测试跟踪看板——从策划表一键导入每赛季新增宠物，按五大类 34 项标准检查清单逐项打勾，支持头像粘贴放大、Bug 单关联跳转、获取方式下拉选择和实时备注，替代传统 Excel 测试表。

## 功能特性

- 📋 **34 项标准化检查清单**，分为策划配置、美术资源、界面功能、投放验证、特殊专项 5 大类
- 🐾 **一键导入**版本新增宠物外观（从策划配置表读取）
- 📷 **头像粘贴**（Ctrl+V）与放大查看（灯箱模式）
- 🔗 **Bug 工单关联**，自动识别 `#工单号` 并跳转
- 📝 **单元格备注**与右键快捷菜单
- 🔄 **多人协同**，数据实时同步到服务端
- 📊 **进度统计**，实时显示 Pass / Fail / 待测比例

## 快速启动

### 环境要求
- Python 3.7+
- Flask

### 安装依赖
```bash
pip install flask
```

### 启动服务
```bash
cd pet_tracker
python server.py
```

服务默认运行在 `http://0.0.0.0:5210`，局域网内其他人可通过你的 IP 地址访问。

## 使用说明

1. 点击 **「载入版本宠物」** 从策划表导入当前赛季宠物
2. 也可手动 **「新增宠物」** 添加
3. 单击测试格子循环切换状态：`pass → fail → 待测 → N/A → 清除`
4. 右键格子可添加 **备注** 或关联 **Bug 工单**
5. 单击头像选中后 **Ctrl+V** 粘贴截图，双击打开文件选择器
6. 悬浮头像点击 🔍 可放大查看
